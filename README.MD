# Password Manager

Aplicação desenvolvida em Python com Tkinter para gerar e armazenar senhas de forma simples.

## Funcionalidades

* Geração automática de senhas seguras.
* Combinação aleatória de letras, números e símbolos.
* Cópia automática da senha gerada para a área de transferência.
* Armazenamento de credenciais em arquivo local.
* Interface gráfica desenvolvida com Tkinter.

## Tecnologias Utilizadas

* Python
* Tkinter

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Danchristian2004/password-manager.git
```

2. Entre na pasta do projeto:

```bash
cd password-manager
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Execute o programa:

```bash
python main.py
```
